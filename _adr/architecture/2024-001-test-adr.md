## Context

ADR(Architecture Decision Records) 시스템을 블로그에 도입하기 위한 테스트 페이지입니다.

이 페이지는 프론트엔드 검색 인덱스에 포함되지 않으며, URL 직접 접근만 가능합니다.

## Decision

`_adr/` 컬렉션을 Jekyll에 추가하고 MCP 서버(`dgnppr-adr`)를 통해 관리한다.

## Consequences

- ADR 문서는 `/adr/<slug>/` URL로 접근 가능
- 사이트 검색, 사이드바, 네비게이션에 노출되지 않음
- Claude Code에서 `/adr` 스킬로 조회·작성 가능

## Alternatives Considered

1. **Notion 연동**: 외부 서비스 의존성 생김 — 미채택
2. **`_wiki/` 통합**: 검색 인덱스에 노출됨 — 미채택
