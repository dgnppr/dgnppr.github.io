# 과거

우연찮게 오랜만에 나의 블로그에 들어와서 여러 글을 다시 읽어보았다. 
불과 2년 전에 쓴 글만 보더라도 자바, 스프링 등 내부 구현과 시스템 디자인핮 관심이 있었던 편이다. 
2년전만 하더라도 코드를 잘 작성하는 것이 개발자의 전부는 아니지만, 정말 중요한 능력이라고 생각했었다.
GPT의 등장 이후에도 코드 퀄리티나 속도 면에서 온전하게 믿고 사용할 수는 없었어서 보조 수단으로만 사용하였다.
